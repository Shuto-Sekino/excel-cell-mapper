# excel-cell-mapper 設計概要

## プロジェクト概要

`excel-cell-mapper` は、Excelファイルをスキーマ（セルIDマッピング）に基づいてJSONへ変換するライブラリです。
ユーザーはセルIDを使ったDSL（ドメイン固有言語）でマッピングを定義し、Excelの任意のセルの値を構造化されたJSONに変換できます。

## 解決する課題

- Excelベースのデータ入力フォームをシステムに取り込む際の手動変換コストを削減する
- セルの位置に依存した柔軟な構造化データ抽出を可能にする
- ネストしたオブジェクトや配列など複雑なJSON構造もスキーマ定義だけで表現できる

## コアコンセプト

### セルIDによるマッピング

Excelのセル参照（例: `A1`, `B3`, `Sheet2!C5`）をJSONのフィールドにマッピングします。

```
Excel セル ──(schema)──▶ JSON フィールド
```

### スキーマ（Schema）

スキーマはJSONオブジェクトで定義します。

| 記法 | 説明 |
|------|------|
| `{ "fieldName": "B1" }` | 静的キー + セル参照 |
| `{ "A1": "B1" }` | 動的キー（A1の値）+ セル参照（B1の値） |
| `{ "fieldName": { ... } }` | ネストしたオブジェクト |
| `{ "fieldName": ["A1:A10"] }` | セル範囲から配列 |

## 設計方針

- **シンプル性**: 最小限のスキーマ定義でほとんどのユースケースに対応する
- **型安全性**: TypeScriptの型定義を提供し、スキーマとJSONの型を推論可能にする
- **拡張性**: カスタムトランスフォーマーやバリデーターを追加できる
- **シート対応**: 複数シートにまたがったマッピングをサポートする
